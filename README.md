# 🏥 Healthcare Data Analytics

An end-to-end Healthcare Data Analytics project focused on predicting hospital readmission risk using data analysis, machine learning, SQL, and interactive Power BI dashboards.

---

## 🎯 Project Objectives

- Analyze patient healthcare records
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Build predictive machine learning models
- Develop interactive Power BI dashboards
- Generate actionable healthcare insights

---

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- SQL
- Scikit-learn
- Matplotlib
- Plotly
- Power BI
- Git & GitHub
